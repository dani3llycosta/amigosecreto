## Desafio: Amigo Secreto

Este projeto foi desenvolvido como parte de um desafio proposto pela Alura, com o objetivo de aplicar e aprimorar meus conhecimentos. O desafio consiste em criar uma aplicação para realizar sorteios de "Amigo Secreto" entre amigos, com a funcionalidade de exibir e sortear nomes, além de permitir a reinicialização do sorteio.

### Funcionalidades

- **Cadastro de Amigos**: O usuário pode adicionar nomes à lista de amigos.
- **Sorteio Aleatório**: Ao clicar no botão de "Sortear amigo", o sistema sorteia aleatoriamente um amigo da lista e exibe o nome sorteado.
- **Desabilitar Input**: Após o sorteio, o campo de digitação é desabilitado e ocultado para evitar novos registros.

### Tecnologias Utilizadas

- **HTML**: Para a estruturação da página.
- **CSS**: Para estilizar a página e tornar a interface amigável.
- **JavaScript**: Para adicionar interatividade e lógica ao projeto, como o sorteio aleatório e a manipulação de elementos HTML dinamicamente.

### Como Usar

1. **Adicionar Amigos**: No campo de texto, insira o nome de um amigo e clique em "Adicionar" para incluir na lista.
2. **Sortear Amigo**: Após adicionar amigos, clique em "Sortear amigo" para ver quem foi o amigo sorteado.

## Considerações Finais

Este projeto foi uma ótima oportunidade explorar o uso de JavaScript para manipulação dinâmica de elementos da página. A decisão de adicionar a funcionalidade de desabilitar o campo de input foi uma forma de melhorar a experiência do usuário, evitando a adição de novos amigos após o sorteio. Pretendo adicionar novas funcionalidades, como um botão para reiniciar o jogo, futuramente.
